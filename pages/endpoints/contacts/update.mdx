---
title: 'Update Contact'
openapi: 'POST /workspaces/{workspace}/contacts/{contact}'
---

### JSON Example

```json
{
  "name": "John Smith",
  "email": "johnsmith@example.com",
  "gender": "male",
  "status": true
}
```

### Multipart Form Data Example

```
PUT /contacts/123
Content-Type: multipart/form-data

name=John Smith
email=johnsmith@example.com
gender=male
status=1
avatar=@/path/to/avatar.png
contact_groups[]=10
contact_groups[]=12
```
